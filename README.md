# DemoClockApp
模仿製作 iOS Clock App
  
## 功能
- 世界時鐘：查詢不同時區目前時間
- 鬧鐘：設定每週特定幾天的時間鬧鐘，可選擇自訂音效
- 碼錶：可分圈計時，以及分別以紅色和綠色標示花最多和花最少時間的圈數
- 計時器：設定倒數計時，可選擇自訂音效
  
## 技術
- 透過第三方套件 ***CodableCSV*** 解析csv資料
- 實現本地推播通知 ***Local Notification***
- 自訂型別編碼解碼Data，進一步讀寫檔
- 傳遞資料
  - ***IBSegueAction***
  - ***UnwindSegue*** 搭配 ***prepare***
- 協定委派模式 ( ***protocol delegation pattern*** )
- 播放音樂 ***AVPlayer***
- ***Timer***

- 熟悉 ***UITableViewController***
- 熟悉 UI 元件
  - ***UITableView***
  - ***UIPickerView***
  - ***tab item button***
